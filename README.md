## Backend API

Framework: Nest.JS <br/>
Port: 5000 <br />
Version: 1.0.0 <br />
Perfix: /api <br />
PackageManager: pnpm <br />

## Getting Started

```bash
pnpm run dev
```

starting on: http://localhost:5000/api

### GET /api/hello

Returns a JSON object with a message.

```json
{
  "message": "Hello World!"
}
```
